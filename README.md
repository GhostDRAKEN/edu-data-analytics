# Educational Data Analytics

This project aims to analyze educational performance data
using Python for data analysis and Django for API exposure.

## Problem
How can we analyze and visualize student performance
to identify trends and potential difficulties?

## Data
CSV files containing grades, attendance, and semesters.

## Stack
- Python (pandas, numpy)
- Django + Django REST
- PostgreSQL
- Web dashboard

## Goals
- Practice data analysis
- Build a backend API
- Create a clear dashboard
- Deploy on the cloud
